# Sales Performance Analysis Using PostgreSQL

## Objective
This project analyzes sales data using PostgreSQL to identify revenue trends, top-performing products, and country-level sales performance. The objective is to demonstrate practical SQL skills commonly used by data analysts in real business scenarios.

## Database Schema
The database consists of four related tables:
- customers: customer-level information
- products: product catalog and categories
- orders: order-level data
- order_items: transactional sales data

## Tools & Technologies
- PostgreSQL
- SQL (JOIN, GROUP BY, aggregations)
- VS Code

## Key Analyses
- Total revenue by country
- Top-performing product by revenue
- Revenue contribution by product category

## Key Findings
- The USA generated the highest total revenue
- Electronics is the top-performing category
- Laptop is the highest revenue-generating produc

## How to Run This Project
- Create the database
- Run schema.sql to create tables
- Run insert_data.sql to insert sample data
- Run queries.sql to perform analysis
